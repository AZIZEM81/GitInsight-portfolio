About GitInsight

GitInsight was born out of a personal need to showcase my GitHub activities more effectively. As a developer, I realized that while GitHub provides valuable information, there was an opportunity to create a tool that offers deeper insights and more visually appealing representations of one's coding journey.

This project is part of my portfolio for Holberton School, demonstrating my skills in full-stack development, data visualization, and API integration.
